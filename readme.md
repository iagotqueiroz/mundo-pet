Visão geral da tela

Pelo que aparece no layout, eu separaria assim:

topo com a marca → header
conteúdo principal da agenda → main
título + subtítulo da página → section com h1 e p
filtro de data → form
blocos Manhã / Tarde / Noite → section
lista de agendamentos dentro de cada bloco → ul e li
cada agendamento → article ou li
botão novo agendamento → button
